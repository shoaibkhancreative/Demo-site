Drop your chapter thumbnails here named exactly:
chapter-1.jpg ... chapter-7.jpg   (16:10 recommended, e.g. 1280x800)

If a file is missing (or fails to load), each chapter automatically shows a
generated placeholder instead: its number in large serif type over a small
deterministic sparkline unique to that chapter. So the grid looks intentional
either way — real photos are an optional upgrade, not something the layout
depends on.

Any photo you do add is automatically given the same warm duotone treatment
(see the `.thumb img` filter in src/styles/global.css) so it matches the
site's palette without manual editing.
